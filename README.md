# vue_clock
時間

# 範例
https://bonnie19931112.github.io/vue_clock/

# CDN

```
 <script src="https://unpkg.com/vue@next"></script>
 <script src="https://bonnie19931112.github.io/vue_clock/main.js"></script>
```

# 說明

想要顯示時鐘效果元素添加 ID 與資料

```
<div id="clock">
     {{ h }} : {{ m }} : {{ s }}
</div>
```
